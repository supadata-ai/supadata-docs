## Installation

The Supadata integration is available as a module in Make. To install:

1. Get your API key from [Supadata Dashboard](https://dash.supadata.ai)
1. Open your Make dashboard
1. Create a new scenario and search for "Supadata"
1. Click **Install**
1. Add Supadata API key when creating a connection
