## Installation

The Supadata integration is available as an app in Zapier. To install:

1. Get your API key from [Supadata Dashboard](https://dash.supadata.ai)
1. Open your Zapier dashboard
1. Create a new Zap and search for "Supadata"
1. Select Supadata from the app list
1. Add Supadata API key when connecting your account
