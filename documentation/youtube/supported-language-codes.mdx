## Supported YouTube languages

The `lang` parameter value designates the desired language for video transcripts. By specifying a particular language code in the YouTube Transcripts API requests, you can indicate your preference for the language of the transcript.

If the preferred language is not available yet, a default language will be used.

You can use a [dedicated endpoint](/documentation/youtube/get-transcript-translation) to perform translation.
